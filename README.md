# CoachReflection

AI-powered coaching journal for football coaches. Reflect on sessions, get AI insights, track progress, and build confidence in your coaching identity.

**Platforms:** Web, iOS, Android

## Getting Started

### Web Development

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

### Mobile Development

CoachReflection supports iOS and Android via Capacitor.

**Quick start:**
```bash
# One-time setup
npm run cap:init

# Test on iOS
npm run ios:dev

# Test on Android
npm run android:dev
```

- [MOBILE-QUICK-START.md](MOBILE-QUICK-START.md) - Quick reference
